# Usage
<!-- usage -->
```sh-session
$ npm install -g -v-
$ -v- COMMAND
running command...
$ -v- (-v|--version|version)
-v-/1.2.7 win32-x64 node-v14.17.5
$ -v- --help [COMMAND]
USAGE
  $ -v- COMMAND
...
```
